# BCRM 美容院CRM系统开发计划

**文档版本**：V1.0  
**编写日期**：2026年3月13日  
**项目周期**：12周

---

## 一、项目概述

### 1.1 项目信息

| 项目 | 说明 |
|------|------|
| 项目名称 | BCRM 美容院客户关系管理系统 |
| 项目类型 | Web应用系统 |
| 开发模式 | 独立开发者全栈开发 |
| 开发周期 | 12周（约3个月） |
| 原型设计 | D:\develope\BCRM\prototype\ |

### 1.2 功能模块

| 模块 | 优先级 | 说明 |
|------|--------|------|
| 客户管理 | P0 | 客户档案、来源追踪、标签分类 |
| 预约管理 | P0 | 预约登记、日历排程、状态流转 |
| 项目管理 | P0 | 服务项目、套餐配置、价格管理 |
| 项目消耗 | P0 | 次卡/储值卡管理、消费记录 |
| 进销存 | P1 | 产品库存、出入库、预警 |
| 会员管理 | P1 | 等级权益、积分预留 |
| 客户维系 | P1 | 生日关怀、消费关怀、沉睡唤醒 |
| 消息提醒 | P1 | 站内消息、短信/微信预留 |
| 数据分析 | P0 | 营收统计、客户分析、员工绩效 |
| 系统管理 | P0 | 用户权限、字典配置、日志 |

### 1.3 技术栈

**后端**
- JDK 17+
- Spring Boot 3.x
- MySQL 8.0
- MyBatis-Plus
- Druid 连接池
- JWT Token 认证

**前端**
- Node.js 18+
- Vue 3 + Composition API
- Element Plus
- Pinia 状态管理
- Vue Router
- Axios
- ECharts 图表库

**工具**
- Maven 3.8+
- Git
- VS Code / IntelliJ IDEA

| 阶段 | 周期 | 主要工作 | 里程碑 |
|------|------|---------|--------|
