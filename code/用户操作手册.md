# BCRM 美容院客户关系管理系统
## 用户操作手册

---

## 一、系统概述

BCRM是一套专为美容院设计的客户关系管理系统，主要功能包括：

- **客户管理**：客户档案、标签分类、来源追踪
- **预约管理**：预约登记、日历排程、状态管理
- **项目管理**：服务项目、套餐配置、价格管理
- **消费管理**：开单消费、会员卡管理、充值记录
- **进销存**：产品库存、出入库管理
- **客户维系**：生日关怀、消费关怀、沉睡唤醒
- **数据分析**：营收统计、客户分析、员工绩效
- **系统管理**：用户权限、数据字典、操作日志

## 四、客户管理

### 4.1 客户列表

**访问路径**：客户管理 → 客户列表

**功能说明**：
- 查看所有客户信息
- 支持按姓名、手机号、会员等级筛选
- 支持关键词搜索

### 4.2 新增客户

1. 点击【新增】按钮
2. 填写客户信息：
   - **基本信息**：姓名、手机号（必填）、性别、生日
   - **来源信息**：客户来源、介绍人
   - **其他信息**：备注、标签
3. 点击【确定】保存

### 4.3 客户详情

点击客户姓名或【查看】按钮，进入详情页面：

- **基本信息**：客户档案
- **消费记录**：历史消费明细
- **会员卡**：持有的会员卡信息
- **预约记录**：预约历史

### 4.4 客户分类

系统自动分类：

| 类型 | 定义 |
|------|------|
| 潜在客户 | 已登记但未消费 |
| 新客户 | 首次消费30天内 |
| 老客户 | 消费超过3次且最近消费在90天内 |
| VIP客户 | 累计消费满5000元或单次消费满2000元 |
| 沉睡客户 | 90天未消费的老客户 |

## 五、预约管理

### 5.1 日历视图

**访问路径**：预约管理

**功能说明**：
- 月视图展示所有预约
- 不同颜色表示不同状态
- 点击日期查看当天预约

### 5.2 新增预约

1. 点击日历上的日期或【新增】按钮
2. 填写预约信息：
   - **客户**：选择客户（支持搜索）
   - **服务项目**：选择服务项目
   - **预约时间**：选择日期和时段
   - **美容师**：选择服务人员
   - **备注**：特殊说明
3. 系统自动检测时间冲突
4. 点击【确定】保存

### 5.3 预约状态

| 状态 | 说明 |
|------|------|
| 待确认 | 新建预约，等待确认 |
| 已确认 | 确认预约，等待服务 |
| 已完成 | 服务完成 |
| 已取消 | 客户取消 |

### 5.4 状态操作

- **确认**：待确认 → 已确认
- **完成**：已确认 → 已完成（同时生成消费记录）
- **取消**：任意状态 → 已取消

## 六、项目管理

### 6.1 服务项目

**访问路径**：项目管理 → 服务项目

**功能说明**：
- 管理所有服务项目
- 支持分类筛选

- 储值套餐：充值送金额

## 七、消费管理

### 7.1 开单消费

**访问路径**：消费管理 → 消费记录 → 【新增】

**操作步骤**：
1. 选择客户
2. 选择消费项目（服务项目或套餐）
3. 选择支付方式：
   - 现金/微信/支付宝
   - 会员卡扣款（次卡扣次/储值扣款）
4. 确认金额
5. 完成开单

### 7.2 会员卡管理

**访问路径**：消费管理 → 会员卡管理

**卡类型**：
- **次卡**：限定某项目使用次数
- **储值卡**：预存金额，消费时扣款

**操作**：
- 查看卡片余额/次数
- 冻结/解冻卡片
- 作废卡片

### 7.3 客户充值

**访问路径**：消费管理 → 客户充值

**操作步骤**：
1. 选择客户
2. 输入充值金额
3. 选择支付方式
4. 确认充值

### 7.4 会员等级

**访问路径**：消费管理 → 会员等级

**等级规则**：

| 等级 | 累计消费 | 折扣 |
|------|---------|------|
| 普通会员 | 0-999元 | 9.5折 |
| 银卡会员 | 1000-2999元 | 9折 |
| 金卡会员 | 3000-5999元 | 8.5折 |
| 钻石会员 | 6000元以上 | 8折 |

## 十三、常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码。

### Q2: 预约时间冲突怎么办？

系统会自动检测时间冲突，如提示冲突请选择其他时间。

### Q3: 会员卡余额不足怎么办？

可以引导客户充值或使用其他支付方式。

### Q4: 如何导出数据？

在数据分析页面，点击【导出】按钮可导出Excel报表。
